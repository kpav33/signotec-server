signoPAD-API/Web Demo

This demo shows how to use the signoPAD-API/Web to easily capture signatures using MS Edge, Mozilla Firefox, Google Chrome, Opera...
To sign click the 'Sign' button.
The signing process will be shown in real time in the 'Live Signature' element.
The signature image will be shown in the 'Signature Image' element.
The SignData is as a string (Base64 encoded) and will be shown in the 'SignData as Base64 string' element.
The RsaSignature is as a string (Base64 encoded) and will be shown in the 'RsaSignature as Base64 string' element.
The 'Pad Type' element shows the pad type for the connected signotec pad.
The 'Serial Number' element shows the serial number for the connected signotec pad. It is only unique within its model range.
The 'Firmware Version' element shows the firmware version for the connected signotec pad.
The 'RSA Support' element shows the RSA encryption capability for the connected signotec pad.
The 'Biometry Cert. ID' element shows the biometry encryption certificate ID for the connected signotec pad.
The 'RSAScheme' element shows the RSA scheme of the RsaSignature from connected signotec pad.
The 'Signature Cert. ID' element shows the signature certificate for the connected signotec pad.
To clear refresh the browser (F5).

Please note that the sign data, the signature image and the RSA signature should be send back to the web application for further processing.
